**LargeRaids** is a vanilla Spigot game experience enhancement plugin for [raids](https://minecraft.fandom.com/wiki/Raid), which are added to the game in the Minecraft's 1.14 Village & Pillage Update. It expands the raid's mechanism to accommodate for the multiplayer environment with higher difficulty, higher bad omen levels, more raiders, more waves and higher rewards.

## Server Requirements

- Version: 1.14.4, 1.15.2, 1.16.5, 1.17.1
